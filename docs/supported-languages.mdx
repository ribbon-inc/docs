---
title: "Supported Languages"
description: "Ribbon API supports multiple languages to accommodate diverse user bases across different regions."
---

Each language code follows the format `{language}-{REGION}` using standard IETF language tags.

| Language Code | Language   | Region/Variant       |
| ------------- | ---------- | -------------------- |
| en-US         | English    | United States        |
| de-DE         | German     | Germany              |
| es-419        | Spanish    | Latin America        |
| hi-IN         | Hindi      | India                |
| ja-JP         | Japanese   | Japan                |
| pt-BR         | Portuguese | Brazil               |
| fr-FR         | French     | France               |
| zh-CN         | Chinese    | China                |
| ko-KR         | Korean     | Korea                |
| vi-VN         | Vietnamese | Vietnam              |
| th-TH         | Thai       | Thailand             |
| ar-AE         | Arabic     | United Arab Emirates |
